Each link below opens the real folder in SharePoint. If your email has been granted access, Microsoft will send you a one-time code to confirm it's you — no separate account needed.

🔒 How access works

  1. Clicking a folder above takes you straight to sharepoint.com — Microsoft's real site, not a copy.
  2. If your email isn't already granted access, ask the folder owner to add you via SharePoint's Share → Specific people.
  3. Once you're on the list, opening the link sends a one-time code to your inbox — verified entirely by Microsoft.
  4. This page never sees your email, password, or code. It's just a set of bookmarks.
Access is managed directly in SharePoint by the document owner, not by this page.
